# @lesto/migrate

> Lesto's migrator — a Tracks-style migration system (ordering, bookkeeping, indexes) over @lesto/db schema-as-value DDL on an injected SQL database.

Part of **[Lesto](https://lesto.run)**, the batteries-included, agent-native fullstack framework.

```bash
bun add @lesto/migrate
```
