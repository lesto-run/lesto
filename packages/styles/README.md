# @lesto/styles

> Lesto's CSS build pipeline (ADR 0037) — a pure buildStyles() over an injected StyleCompiler seam, so an app compiles Tailwind v4 to a served stylesheet with no bespoke build script. The real @tailwindcss engine wiring lives in a separate coverage-excluded edge.

Part of **[Lesto](https://lesto.run)**, the batteries-included, agent-native fullstack framework.

```bash
bun add @lesto/styles
```
