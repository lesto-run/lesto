# create-lesto

> The Lesto project scaffolder — `bun create lesto <name>` / `npm create lesto <name>`.

Part of **[Lesto](https://lesto.run)**, the batteries-included, agent-native fullstack framework.

```bash
bun create lesto my-app
# or: npm create lesto my-app

cd my-app
bun install
bun run dev
```

Every generated app ships an `AGENTS.md`, a `CLAUDE.md`, and a
`.claude/skills/lesto` skill, so a coding agent can drive it from the first commit.
