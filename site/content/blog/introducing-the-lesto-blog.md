# Introducing the Lesto blog

This is where we'll write about building Lesto — the batteries-included,
agent-native, full-stack TypeScript framework.

Expect three kinds of posts:

- **Deep dives** into how the batteries work — the durable queue on the database,
  browser-to-server tracing with no OpenTelemetry dependency, the one-substrate
  bet, and the agent control plane.
- **Release notes**, summarizing what's new each version. The full, generated log
  lives on the [changelog](/changelog).
- **The thinking** behind the design decisions — many of which started as ADRs in
  the repo.

If you're new here, start with [Why Lesto](/why-lesto) for the positioning, then
the [Quickstart](/quickstart) to get an app running in about five minutes.

More soon.
